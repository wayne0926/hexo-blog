# 优点

- [x] 纯静态
- [x] `MD5` 加密，不可逆
- [x] 登陆后反复后台验证，比较安全
- [x] 验证账户密码直接从表单获取而不赋值
- [x] 虽是手动添加用户名密码，却很方便
![代码片段](https://cdn.jsdelivr.net/gh/wayne0926/myphoto/img/Snipaste_2020-11-15_01-18-28.jpg)

# 缺点
- [x] ~~`JavaScript` 里是以赋值的方式获取表单，有可能会被劫持然后直接把存在的 `MD5` 写进去~~
- [ ] 后台反复验证可能会导致性能下降，设备运行卡顿
- [ ] 输入完账户密码后回车来进行登录会有很大的延迟
